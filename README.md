# node-ccmirror
ComputerCraft Mirror Client

# Installation
Installation can be done via the following commands
```sh
git clone https://github.com/RoDaBaFilms/node-ccmirror.git
cd node-ccmirror
npm i
npm i -g .
```

# Usage
Grab your terminal and execute the following command
```sh
ccmirror <code>
```
